# AI Image Generator (.NET)

A demo project exploring how ASP.NET Core Minimal APIs can integrate
with open-source text-to-image generation services.

## Features
- Prompt-based AI image generation
- ASP.NET Core Minimal API
- Swagger image preview using HTML response
- Demo-safe request throttling
- No local AI model installation required

## Tech Stack
- .NET (ASP.NET Core)
- Minimal APIs
- Swagger (Swashbuckle)

## Notes
⚠️ Uses public inference endpoints for experimentation purposes.
Not intended for production or high-traffic usage.

## Motivation
Built as a learning project to understand how AI capabilities
can be integrated into modern .NET backend systems.
